# MVC Report
The repository contains all the files for the web application report. This is created as part of the Objektorienterade webbteknologier (mvc) course at Blekinge Tekniska HÃ¶gskolan. Object-oriented PHP and Symfony framework are used to create the web application.
To clone this repository run ```git clone https://github.com/hafsaChy/report```.
To run the app locally run ```php -S localhost:8888 -t public```. Then, the app can be seen in ```http://localhost:8888/``` from the browser.
